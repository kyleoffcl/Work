# Grow With Us Agency — Website

A modern, single-page marketing site for **Grow With Us Agency**, a done-for-you
digital marketing agency. Built as a fast, fully responsive static site with rich
gradients, animated background orbs, scroll-reveal effects, and animated stat
counters.

## Sections
- Hero with animated gradient glow
- Animated statistics counters
- Services (social growth, lead gen, ads, web/app dev, content, PR)
- Scrolling capabilities marquee
- Tiered pricing packages
- About / why us
- Testimonials
- **Locations:** Dubai, United Arab Emirates & Miami Beach, Florida
- Contact form + direct email: **info@growwithusagency.me**

## Run locally
Just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Files
- `index.html` — markup
- `styles.css` — design tokens, layout, animations
- `script.js` — nav, scroll reveal, counters, form handling

No build step or dependencies required.
